# Internship Task Tracker

## Overview
A mobile Android application built with Java and Android Studio that helps 
users track and manage internship-related tasks. Users can add, edit, delete, 
and mark tasks as completed. All tasks are stored locally using the Room 
database so data persists between app sessions.

## Demo Video
[Watch the demo on YouTube](YOUR_YOUTUBE_LINK_HERE)

## Features
- Add new tasks with a title, description, and due date
- Edit existing tasks
- Delete tasks with a confirmation dialog
- Mark tasks as completed with a strikethrough effect
- Tasks sorted by due date
- Persistent local storage using Room database

## Technologies Used
- Java
- Android Studio
- Room Database (SQLite)
- RecyclerView
- LiveData and ViewModel (Android Architecture Components)

## How to Run
1. Clone this repository
2. Open in Android Studio
3. Connect an Android device or launch an emulator
4. Click the Run button to build and launch the app
